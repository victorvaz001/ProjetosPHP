# Site - cadsatro somente por convite <br>

## Objetivo
* Projeto criado para criar um site de somente convidados<br>

## Funções

* Login de acesso com sessão<br>
* INSERT<br>
* DELETE<br>
* UPDATE<br>
* SELECT


## Tecnologias

* PHP Version 7.4.8<br>
* HTML<br>
* Bootstrap<br>
* MySQL

## Testes<br>
